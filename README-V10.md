# REPORT.IA RCV — Intelligence Decision 10.0

Construida sobre la versión 9.1 estable.

## Nuevas funciones
- Decision Center con Executive Score 0–100.
- Panel automático de prioridades ejecutivas.
- Drill-down ejecutivo por gerencia.
- Comparativos guardados en el navegador.
- Resumen inteligente orientado a decisiones.
- Conserva Alertas, Copiloto RCV, Comparador, Modo Presentación y exportaciones de la 9.1.

## Importante
Esta versión agrega una capa de análisis no destructiva. No cambia la lógica original de carga, procesamiento ni autenticación.

## Instalación
Sube todo el contenido del ZIP a la raíz de tu repositorio de GitHub Pages y actualiza con Ctrl+F5.
