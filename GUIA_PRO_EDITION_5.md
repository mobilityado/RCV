# REPORT.IA RCV · Pro Edition 5.0

## Centro de Prioridades
Ordena automáticamente gerencias críticas, focos de atención, productividad y oportunidades.

## Copiloto REPORT.IA
Responde preguntas básicas sobre gerencias, presupuesto, XPV y prioridades usando únicamente la información procesada en el navegador.

## Cierre Ejecutivo
Resume el periodo y muestra el Pulso RCV junto con la conclusión general.

## Modo Comité
Oculta carga, navegación y acciones secundarias para mostrar únicamente:
1. Resumen Ejecutivo
2. Prioridades
3. Gerencias
4. Tendencias
5. Cierre Ejecutivo

Presiona Escape para salir del Modo Comité.

## Publicación
Sube el contenido de la carpeta REPORTIA_RCV a la raíz de tu repositorio y activa GitHub Pages.
